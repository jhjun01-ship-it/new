🚀 GitHub Gist로 대시보드 공유하기

단계별 상세 가이드 - 초보자도 5분만에 완성!

⏱️ 소요시간: 3-5분 💰 비용: 무료 🌍 접근성: 전세계

📋 사전 준비사항

필요한 것

  • GitHub 계정 (무료)
  • 인터넷 브라우저
  • HTML 코드 (아래 제공)

GitHub 계정이 없다면?

  1. 1. github.com 접속
  2. 2. "Sign up" 클릭
  3. 3. 이메일, 비밀번호, 사용자명 입력
  4. 4. 이메일 인증 완료
1

GitHub Gist 페이지 접속

🌐 접속 URL

https://gist.github.com

GitHub Gist 메인 페이지 스크린샷

(검은색 상단바, "Create a new gist" 버튼 보임)

⚠️ 주의: GitHub에 로그인되어 있어야 합니다. 로그인하지 않으면 "Sign in to GitHub" 버튼이 보입니다.

2

새 Gist 생성

페이지 상단의 "Create a new gist" 버튼을 클릭합니다.

"Create a new gist" 버튼 위치

(페이지 우상단, 녹색 버튼)

✅ 성공 확인: 새로운 페이지에서 파일 이름 입력창과 코드 편집기가 나타납니다.

3

파일 이름 설정

상단의 "Filename including extension..." 입력창에 다음과 같이 입력합니다:

pleats_mood_dashboard.html

💡 파일명 규칙:

  • 반드시 .html 확장자 포함
  • 공백 대신 언더바(_) 또는 하이픈(-) 사용
  • 한글 사용 가능하지만 영문 권장

파일명 입력창

(상단의 긴 입력창, 플레이스홀더 텍스트 있음)

4

HTML 코드 붙여넣기

아래의 큰 텍스트 편집 영역에 HTML 코드를 붙여넣습니다.

⚠️ 중요: 코드를 복사할 때 Ctrl+A (전체선택) → Ctrl+C (복사) → Ctrl+V (붙여넣기) 를 사용하세요.

<!DOCTYPE html> <html lang="ko"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>플리츠무드 PRO 글로벌몰 매출 분석</title> <script src="https://cdn.jsdelivr.net/npm/chart.js"></script> <script src="https://cdn.tailwindcss.com"></script> <!-- 전체 코드는 너무 길어서 축약 표시 --> <!-- 실제로는 이전에 제공한 전체 HTML 코드를 복사해야 합니다 --> </head> <body> <!-- 대시보드 내용 --> </body> </html>

✅ 확인 방법: 코드를 붙여넣으면 텍스트에 색상이 입혀집니다 (문법 하이라이팅).

5

Gist 설정 및 생성

📝 설정 옵션

✅ Public Gist (권장)
  • • 누구나 링크로 접근 가능
  • • 검색 엔진에서 찾을 수 있음
  • • 완전 무료
🔒 Secret Gist
  • • 링크를 아는 사람만 접근
  • • 검색 엔진에 노출 안됨
  • • 더 높은 프라이버시

🎯 생성 버튼

또는

💡 추천: 외부 공유 목적이라면 "Create public gist"를 선택하세요.

6

공유 링크 생성

Gist가 생성되면 새로운 페이지로 이동합니다. 이 페이지의 URL이 기본 Gist 주소입니다.

📋 URL 형태

https://gist.github.com/[사용자명]/[고유번호]

예: https://gist.github.com/john123/1a2b3c4d5e6f7g8h9i0j

🌐 웹페이지로 변환

대시보드를 웹페이지로 보려면 URL 끝에 /raw를 추가합니다:

변환 과정:
기존 https://gist.github.com/john123/1a2b3c
↓
변환 https://gist.github.com/john123/1a2b3c/raw

🎉 완성! /raw가 추가된 URL을 누구에게든 공유하면 바로 대시보드를 볼 수 있습니다!

🔧 문제 해결 가이드

❌ 페이지가 비어있게 나올 때

원인: HTML 코드가 잘못 복사되었거나 파일 확장자가 틀림

해결: 파일명에 .html이 포함되어 있는지 확인, 코드 전체 재복사

⚠️ 차트가 안 보일 때

원인: 외부 라이브러리 로딩 실패 또는 JavaScript 오류

해결: 페이지 새로고침(F5), 다른 브라우저에서 시도

ℹ️ 링크를 클릭해도 안 열릴 때

원인: /raw 추가 안함 또는 Gist가 Private

해결: URL 끝에 /raw 추가, Public Gist로 재생성
